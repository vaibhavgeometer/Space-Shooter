# Space Shooter

A simple space shooter game built with Pygame, created using Google Antigravity and Gemini (Vibe Coding).

## Installation

```bash
pip install pygame
```

## Usage

```bash
python main.py
```

## Features

- Player movement with arrow keys
- Shooting with spacebar
- Collision detection
- Score tracking
- High score saving
- Sound effects
- Game over screen
